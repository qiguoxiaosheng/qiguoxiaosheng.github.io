Idea搭建Flink本地开发环境

工欲善其事，必先利其器！打算学习Flink，撸码工具很重要，本文主要记录如何使用IDEA搭建Flink本地代码开发、调试环境，编程语言以Java为例。

1、创建工程

在IDEA里依次选择

New Project –> Maven –> Create from archetype，然后选择flink-quickstart-java，剩下的就是输入Name，groupId，ArtifactId等信息。如果此处没有flink-quickstart-java骨架，选择右上角的“Add Archetype”，填入相关信息。

2、本地开发、调试

创建好的工程结构如下图所示：

接下来，就可以在里面开心的写代码了^_^。一顿操作之后，习惯性右击–>Debug。然鹅，出Exception了。

麻溜打开了bing，尝试各位大佬的宝典，经测试，下面步骤可行。

Project Settings –>Dependencies选择添加Library，然后分别选择Flink发行包中的lib、opt目录，并添加ScalaSDK。

完成后，重新Debug，熟悉的Hello World又出现了。
